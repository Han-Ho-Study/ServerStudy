x64환경에서 사용 -> 포인터 64비트(8바이트)
0x000000012345678
0x000000112345678
0x000001012345678

printf(%x, bufferName) -> 중복이 됨, Array[3] 보장! Array[6] -> 1A34F678 -> int형으로 표현 -> 32비트(4바이트)
printf(%p, bufferName) -> 중복이 안 됨
 
X	int값을 부호없는 16진수로 출력  10~15은  'A'~'F'로 표시
x	int값을 부호없는 16진수로 출력  10~15은  'a'~'f'로 표시
p	포인터값을 16진수로 출력
